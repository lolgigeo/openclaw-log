## 🚀 服务管理

### 启动服务
```bash
cd ~/docs-hexo
./start-hexo.sh
```

### 重新生成后启动
```bash
./start-hexo.sh --rebuild
```

### 停止服务
```bash
pkill -f 'http-server.*-p 80'
```

### 查看日志
```bash
tail -f ~/docs-hexo/server.log
```

## ✍️ 添加新文档

### 标准流程

1. **创建新文档**
```bash
cd ~/docs-hexo
hexo new "文档标题"
```

2. **编辑Front Matter**
```markdown
---
title: 文档标题
date: 2026-02-06 15:30:00
categories: 分类名称
tags:
  - 标签1
  - 标签2
toc: true
---
```

3. **编写内容**

4. **生成并部署**
```bash
hexo clean
hexo generate
./start-hexo.sh --rebuild
```

### 自动化（按AGENTS.md规则）

✅ **已整合到文档库自动化流程**

当生成新的md文档时：
1. 添加Front Matter
2. 复制到 `~/docs-hexo/source/_posts/`
3. 执行 `hexo generate`
4. 重启服务
